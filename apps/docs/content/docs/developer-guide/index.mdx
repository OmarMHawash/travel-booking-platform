## Welcome, Developer!

This guide is your central resource for understanding how to work on the Travel Booking Platform. It contains everything you need to know about the project's structure, coding conventions, key libraries, and best practices.

Following these guidelines will help you contribute effectively and ensure the codebase remains clean, consistent, and maintainable.

<Callout>
  Before diving in, make sure you have successfully completed the steps in the
  **[Getting Started](/docs/getting-started)** section to set up your local
  environment.
</Callout>
